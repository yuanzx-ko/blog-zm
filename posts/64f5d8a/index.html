搭建博客教程

2023/10/20 教程 博客 编程

我终于拥有了属于自己的博客!

教程来喽

  • 第一步，下载termux并安装必要的软件包
pkg install nodejs
pkg install git

  • 第二步,创建并配置GitHub仓库
    在GitHub上创建一个新的仓库，并复制该仓库的URL

  • 第三步,安装hexo
    在Termux中运行以下命令安装Hexo

npm install -g hexo-cli
  • 第四步，初始化Hexo博客
    创建一个新目录并进入该目录，然后运行以下命令初始化Hexo博客
hexo init myblog
cd myblog
npm install
  • 配置Hexo博客
    打开Hexo博客根目录中的_config.yml文件，并进行以下配置

修改deploy部分的type为git

修改deploy部分的repo为之前复制的GitHub仓库的URL

指定deploy部分的branch为gh-pages

如果想要其他主题的话,请自行安装，安装好后,按照主题文档完成配置

  • 第六步，生成静态文件
hexo generate

如果生成失败的话,请检查配置,也可以在评论区告诉我报错信息,我给你说说什么原因,静态文件也有可能不完整

  • 第七步，部署到GitHub

    这一步容易出错,在termux配置好github用户信息,生成并使用ssh密钥,再安装git部署插件,就可以运行以下命令部署了

hexo deploy

如果在本地修改了博客，请再次运行hexo generate、hexo deploy命令
