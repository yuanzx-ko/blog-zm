live2d看板娘教程

2023/10/21 教程 博客 美化

添加live2d看板娘可能会导致博客访问速度慢…

可爱的live2d看板娘教程来喽

  • 第一步,安装live2d模块
npm install --save hexo-helper-live2d

  • 第二步,下载live2d模型
    新建一个live2dModel文件夹,将下载的live2d模型放进去

  • 第三步，在站点配置文件中新增如下内容

live2d:
  enable: true
  scriptFrom: local
  pluginRootPath: live2dw/
  pluginJsPath: lib/
  pluginModelPath: assets/
  tagMode: false
  debug: false
  model:
    use: <u>live2d-widget-model-hibiki</u> #下载的模型名字
display:
   position: right
   width: 150
   height: 300
   mobile:
     show: true  #手机上是否展示

最后再重新生成一下静态文件,部署，就可以看到live2d模型啦
